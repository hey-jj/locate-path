# Changelog

## [0.2.0] - 2026-07-07

### Changed
- File URL working directories now parse raw query strings and fragments as URL components, and encoded `?` and `#` characters stay in the path. (#18)

### Fixed
- File URL working directories now accept any ASCII letter case in the `file://` scheme. (#17)

### Documentation
- The explicit working directory test wording now names explicit directory selection correctly. (#19)
